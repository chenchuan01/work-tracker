# 快速使用指南

## 🎯 核心功能

### 1. 记录工作
- 打开页面，在输入框中输入工作内容
- 按 `Enter` 保存，`Shift+Enter` 换行
- 记录自动保存到本地浏览器

### 2. 加入待办 ✨
- 点击输入框上方的「✅ 加入待办」切换模式
- 输入待办事项内容
- 按 `Enter` 加入待办列表
- 待办事项会显示在输入框下方

### 3. 待办管理
- **查看待办**：待办列表显示在页面上方，分为「待完成」和「已完成」
- **完成待办**：点击 checkbox 或「✓ 完成」按钮，待办自动转为工作记录
- **取消完成**：在「已完成」折叠区域可取消完成状态
- **删除待办**：点击删除图标删除待办事项

### 4. 查看工作
- **日视图**：查看当天的所有记录
- **周视图**：查看当周的所有记录
- **月视图**：查看当月的所有记录
- 使用左右箭头切换日期
- 点击「今天」快速返回当前日期

### 5. 管理记录
- ⭐ **标记重点**：点击星星图标标记重要工作
- ✏️ **编辑**：点击编辑图标修改内容
- 🗑️ **删除**：点击删除图标删除记录

### 6. AI 总结
- 点击「✨ AI 总结」按钮
- 自动生成 100 字以内的工作简报
- 点击「📋 复制」复制到剪贴板
- 可直接用于周报/月报汇报

### 7. 设置
- 点击右上角 ⚙️ 图标
- 配置 OpenAI API Key（可选）
- 不配置时使用模拟数据

## 💡 使用技巧

### 工作流程推荐

```
早上 9:00  →  列出今天要做的事（加入待办）
            - 参加产品评审会议
            - 修复用户反馈的 bug
            - 编写单元测试

下午 14:00 →  完成一项，勾选「✓ 完成」→ 自动转为工作记录

下午 18:00 →  补充记录其他临时工作

周五 17:00 →  切换到「周视图」→ 点击「AI 总结」→ 复制周报
```

### 最佳实践

1. **早上列计划**：到公司后先把今天要做的加入待办
2. **完成即勾选**：每完成一项立即勾选，获得成就感
3. **随时补充**：临时完成的工作直接记录
4. **标记重点**：把重要工作标记为⭐，生成简报时会优先展示
5. **周报生成**：每周五切换到「周视图」，点击 AI 总结生成周报
6. **月报生成**：月末切换到「月视图」，生成月度工作汇报

## 🔧 技术说明

- **数据存储**：使用浏览器 IndexedDB，数据存储在本地
- **离线可用**：无需联网即可记录工作（AI 总结需要网络）
- **隐私安全**：所有数据存储在本地，不会上传到服务器
